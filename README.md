## Overview
A continuously maintained machine learning knowledge base and technical roadmap.

## Repository Structure
### Foundations（基础）
- `linear-algebra.md` — Linear Algebra / 线性代数
- `probability.md` — Probability & Statistics / 概率统计
- `optimization.md` — Optimization / 优化方法

### Classical ML（传统机器学习）
- `regression.md` — Regression Models / 回归模型
- `svm.md` — Support Vector Machines
- `tree.md` — Decision Trees & Ensembles / 决策树与集成方法

### Deep Learning（深度学习）
- `cnn.md` — Convolutional Neural Networks
- `transformer.md` — Transformer Models
- `training-tricks.md` — Training Tricks / 训练技巧

### ML Systems（工程与系统）
- `data.md` — Data Pipelines / 数据流程
- `training.md` — Training & Evaluation / 训练与评估
- `serving.md` — Deployment & Serving / 部署与服务

## How to Use
- Follow the roadmap from foundations to systems
- Use this repository as a reference or learning guide
